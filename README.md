# Instructions

Close repository, create the entity class and the API as defined below.

All values should be stored in memory.

## Entity definition

Name: `Car`

Fields:
* `STRING` Id (generated)
* `STRING` Name
* `STRING` Brand
* `INTEGER` ModelYear


## API definition

1. List all cars
2. List all cars of a certain brand
3. Add a new car
4. Update a car
5. Delete a car
